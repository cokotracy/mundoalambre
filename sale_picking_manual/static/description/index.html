Create Delivery Orders Manually.

This module allows to select the orderlines and then it creates its Delivery Order.

Key Features:

  • Create Delivery Orders Manually for selected Orderlines.
  • Allow to devide order items in multiple delivery orders.
  • It will stop feature of auto create delivery order from sale order

Sale order: Create Picking button will allow to select order lines


Popup: By-Default it will show all order items on popup


Popup: Remove order items which you not want to consider in delivery order


Picking: Create Delivery order with order items from popup


Popup: Create picking popup will only show products which are not created pickings already


Sale Order: Manual created delivery orders linked with sale order and also delivered quantity will get affected
